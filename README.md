# sketch-application
A Python drawing application designed for STEM note taking and problem solving. 

### Shortcut

* `s` use select tool.
* `p` use pen tool.
* `l` use line tool.
* `r` use rectangle tool.
* `c` use circle tool.
* `e` use elipse tool.
* `a` set anchor.
* `g` go to set anchor.
* `h` hide / show tool bar.
